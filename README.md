# AI Assistant - Web Search & Math Problem Solver

A powerful AI assistant that can search the web, answer questions, and solve math problems — all in one conversation.

## Features

- 🔍 **Web Search**: Get answers from across the internet using Tavily API
- 💬 **Question Answering**: Powered by Groq's Llama 3.1 model
- 🧮 **Math Problem Solving**: Solve calculations, percentages, and equations
- 📚 **Research Assistant**: Find and summarize information from reliable sources
